# InfiniSheet

[GitHub](https://github.com/TheCandidStartup/infinisheet) | [API](https://www.thecandidstartup.org/infinisheet/index.html) 

InfiniSheet is a cloud based, open source, serverless, customer deployed, scalable spreadsheet. Or at least it will be when I've finished. 

This Storybook presents the frontend components from the [react-spreadsheet](?path=/docs/react-spreadsheet--docs) and [react-virtual-scroll](?path=/docs/react-virtual-scroll--docs) packages.
