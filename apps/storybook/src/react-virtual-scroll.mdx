# React Virtual Scroll

[GitHub](https://github.com/TheCandidStartup/infinisheet/tree/main/packages/react-virtual-scroll) | [NPM](https://www.npmjs.com/package/@candidstartup/react-virtual-scroll) | [API](https://www.thecandidstartup.org/infinisheet/modules/_candidstartup_react_virtual_scroll.html) 

React virtual scrolling components for lists and grids. Written in TypeScript using modern React. Scalable to [trillions of rows and columns](?path=/story/react-virtual-scroll-virtualgrid--trillion-square).

Defines the [VirtualList](?path=/docs/react-virtual-scroll-virtuallist--docs) and [VirtualGrid](?path=/docs/react-virtual-scroll-virtualgrid--docs) components. Or build your own by combining [VirtualScroll](?path=/docs/react-virtual-scroll-virtualscroll--docs), [AutoSizer](?path=/docs/react-virtual-scroll-autosizer--docs), [DisplayList](?path=/docs/react-virtual-scroll-displaylist--docs) and [DisplayGrid](http://localhost:6006/?path=/docs/react-virtual-scroll-displaygrid--docs).
